# Setup instructions

1. Open your GitHub profile repository:

   `susilxreactor021-png/susilxreactor021-png`

2. Copy these files into that repository without changing their folders:

   ```text
   README.md
   assets/sushil-banner.svg
   assets/sushil-banner-light.svg
   assets/sushil-hacker-lanyard-preview.png
   assets/sushil-hacker-lanyard-preview.gif
   assets/sushil-trophies.svg
   .github/workflows/snake.yml
   ```

3. Commit and push the files.

4. Open the repository's **Actions** tab.

5. Select **Generate contribution snake** and press **Run workflow**.

6. Wait for the workflow to finish. It creates an `output` branch containing the snake SVG.

7. Refresh your GitHub profile.

## Important checks

- The profile repository name must exactly match your username:
  `susilxreactor021-png`
- The default branch should be `main`.
- In repository settings, GitHub Actions must have permission to write repository contents.
- Replace a project URL in `README.md` if the repository name is different.
